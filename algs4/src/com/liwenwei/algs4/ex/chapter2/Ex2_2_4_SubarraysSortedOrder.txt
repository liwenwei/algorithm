2.2.4 在归并排序中，是否当且仅当两个输入的子数组都有序时，原地归并的方法才能得到正确的结果？

答：是的，当且仅当两个输入的子数组都有序时，才会得到正确的结果。
我们看一下这段代码：
for (int k = low; k <= high; k++) {
	if (i > mid)                    a[k] = aux[j++];
	else if (j > high)              a[k] = aux[i++];
	else if (less(aux[j], aux[i]))  a[k] = aux[j++];
	else                            a[k] = aux[i++];
}
根据左右两个扫描指针i，j 总结起来就是四个条件：
1. 左边取完取右边
2. 右边取完去左边
3. 右边当前元素小于左边当前元素取右边
4. 左边当前元素小于右边当前元素取左边
不管哪种取法，都是左右两边的subarrays的依次顺序取值，所有如果两边数组必须是有序的，输出的结果才会正确
